
# SimplyDoors Costing Tool

A professional, internal pricing calculator for SimplyDoors.

## Features

- **Real-time Cost Calculation**: Instantly calculates internal costs and customer pricing based on selected configurations.
- **Margin Management**: Easily adjust target gross margins to see markup equivalents.
- **Slab Configuration**: Supports various door styles (Woodgrain, Simpson Redi-Prime, Rift White Oak), heights, and types.
- **Add-on Services**: Toggle production services and component materials with automatic cost updates.
- **Drop Ship Logic**: Automatically calculates drop shipment fees based on quantity.
- **Responsive Design**: Works on desktop, tablet, and mobile devices.
- **Brand Identity**: Custom SimplyDoors branding and color palette.

## Tech Stack

- **React 18**: UI Library
- **Vite**: Build tool and dev server
- **Tailwind CSS**: Utility-first CSS framework
- **Lucide React**: Icons
- **Framer Motion**: Animations

## Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/simplydoors-costing-tool.git
    cd simplydoors-costing-tool
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```

## Deployment

This project is ready to be deployed to any static hosting service like Vercel, Netlify, or GitHub Pages.

### Vercel (Recommended)

1.  Push your code to a GitHub repository.
2.  Go to [Vercel](https://vercel.com) and sign up/login.
3.  Click "Add New Project" and import your repository.
4.  Vercel will automatically detect the Vite settings. Click "Deploy".

### Netlify

1.  Push your code to a GitHub repository.
2.  Go to [Netlify](https://netlify.com) and sign up/login.
3.  Click "New site from Git".
4.  Connect your repository.
5.  Build command: `npm run build`
6.  Publish directory: `dist`
7.  Click "Deploy site".
